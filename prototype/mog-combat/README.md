# MOGMAX Combat Browser Prototype

Open `index.html` from this folder, or serve the repo root and visit:

`/prototype/mog-combat/index.html`

The prototype keeps the four-battle ladder:

- Tutorial: shows correct reads for the first turns.
- Easy: removes callouts and adds light feints.
- Medium: adds tempo changes, bait, dodge-only attacks, and jump checks.
- Hard: adds pattern reads, phase-shifted weaknesses, and tighter windows.

Design intent:

- Player attacks use bar, rapid-hit, and ring timing checks.
- Enemy turns use visible windups with parry, dodge, and jump responses.
- Flow, Break, Ward, Cringe, Shield, Scan, weakness, and resistance make repeated one-button play weaker.
- Perfect defensive beats can riposte and build Flow.
- Menu lets you start the whole ladder or jump straight to tutorial, easy, medium, or hard.

Controls:

- `1`-`6`: choose moves.
- `Space`: confirm attack timing or parry.
- `D`: dodge red attacks.
- `J`: jump low attacks.
- `Esc`: open menu.
